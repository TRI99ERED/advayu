# AdvAyu VS Code Theme

AdvAyu is a vibrant and modern Visual Studio Code color theme inspired by Ayu, designed to provide a visually appealing and comfortable coding experience.

## Features
- Carefully crafted color palette for syntax highlighting
- Enhanced readability and focus
- Supports a wide range of languages and file types
- Easy on the eyes for long coding sessions


## Installation

### From the VS Code Marketplace
1. Open Visual Studio Code.
2. Go to the Extensions view by clicking the square icon in the sidebar or pressing `Ctrl+Shift+X`.
3. Search for "AdvAyu" in the Marketplace.
4. Click "Install" on the AdvAyu theme extension.
5. Go to `File` > `Preferences` > `Color Theme` and select "AdvAyu" from the list.

### Manual Installation
1. Download or clone this repository.
2. Open Visual Studio Code.
3. Go to `File` > `Preferences` > `Color Theme` and select "AdvAyu" from the list.
